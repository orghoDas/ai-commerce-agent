# Pricing And Go-To-Market

## Positioning

Sell this as:

```text
An AI sales assistant that answers product questions, checks stock, takes orders, and sends daily inventory reports.
```

Avoid selling it as a generic chatbot. Businesses pay more for operational value than for chat alone.

## Initial Pricing

Starter:

- 1 business
- Website chat
- Up to 500 conversations/month
- Manual catalog
- Daily report

Growth:

- Higher conversation limit
- CSV import
- Human takeover
- WhatsApp channel
- Low-stock alerts

Pro:

- Integrations
- Image search
- Multiple admins
- Multiple branches
- Advanced reporting

## Possible Fees

- Monthly SaaS subscription
- One-time onboarding/import fee
- Usage fee after included conversations
- Pass-through WhatsApp/SMS costs
- Premium integration fee

## Pilot Strategy

1. Pick one niche.
2. Find 5 businesses with messy product messaging.
3. Import their catalog manually.
4. Run the assistant beside a human for 2 weeks.
5. Measure:
   - response time reduction
   - order conversion
   - wrong answer rate
   - human takeover rate
   - admin time saved
